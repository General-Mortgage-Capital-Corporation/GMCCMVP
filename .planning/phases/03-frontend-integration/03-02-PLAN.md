**2. Add renderSimpleMarkdown()** -- In the "Utility Functions" section, add:

A function that takes text string, escapes HTML entities first (`&` -> `&amp;`, `<` -> `&lt;`, `>` -> `&gt;`), then applies basic markdown transforms:
- `**text**` -> `<strong>text</strong>`
- Lines starting with `- ` or `* ` -> `<li>...</li>` wrapped in `<ul>...</ul>`
- Remaining newlines -> `<br>`

This prevents XSS from LLM output while rendering basic formatting.

Returns an HTML string with a `<div class="criteria-grid">` containing one `.criterion-item` per criterion in the tier's criteria array. Each item:

Map criterion status values: "pass" -> STATUS_ICONS.pass, "fail" -> STATUS_ICONS.fail, anything else (including "unverified") -> STATUS_ICONS.unverified.

Returns an HTML string for the Matching Programs section in the modal. Logic:
