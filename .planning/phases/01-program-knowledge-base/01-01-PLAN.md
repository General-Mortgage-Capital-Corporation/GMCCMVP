    - structure_with_llm(markdown_text, program_name) returns a ProgramRules instance
    - structure_with_llm result has program_name matching the input program_name

  <action>
  1. Create `rag/extract.py`:
     - `extract_pdf_to_markdown(pdf_path: str) -> list[dict]`:
       Uses `pymupdf4llm.to_markdown(pdf_path, page_chunks=True)`.
       Returns list of page dicts, each with at least a "text" key containing Markdown.
       Validates pdf_path exists before calling (raise FileNotFoundError if not).

  2. Create `rag/structure.py`:
     - `structure_with_llm(markdown_text: str, program_name: str) -> ProgramRules`:
       Uses `google.genai.Client(api_key=GEMINI_API_KEY)` from rag.config.
       Calls `client.models.generate_content()` with:
       - model=GEMINI_MODEL (from config)
       - contents= a prompt that includes:
         - Program name (from parameter, NOT from PDF)
         - Instruction to extract ALL eligibility tiers/matrices
         - Instruction to identify QM/Non-QM status
         - Instruction: "Only extract values explicitly stated in the guideline. Use null for values not found."
         - The full Markdown text
       - config with response_mime_type="application/json" and response_json_schema=ProgramRules.model_json_schema()
       Returns `ProgramRules.model_validate_json(response.text)`.
       Uses `from rag.schemas import ProgramRules` and `from rag.config import GEMINI_API_KEY, GEMINI_MODEL`.
